# @rolldown/plugin-relay [![npm](https://img.shields.io/npm/v/@rolldown/plugin-relay.svg)](https://npmx.dev/package/@rolldown/plugin-relay)

Rolldown plugin for [Relay](https://relay.dev/) GraphQL transform.

Replaces `graphql` tagged template literals with imports to the compiled artifact files generated by the Relay compiler.

This plugin utilizes Rolldown's [native magic string API](https://rolldown.rs/in-depth/native-magic-string) and is more performant than using [`babel-plugin-relay`](https://npmx.dev/package/babel-plugin-relay) with [`@rolldown/plugin-babel`](https://npmx.dev/package/@rolldown/plugin-babel).

## Install

```bash
pnpm add -D @rolldown/plugin-relay
```

## Usage

```js
import relay from '@rolldown/plugin-relay'

export default {
  plugins: [
    relay({
      // options
    }),
  ],
}
```

## Options

### `language`

- **Type:** `'typescript' | 'javascript'`
- **Default:** `'javascript'`

The language used for generated artifacts. This determines the file extension used when importing compiled artifacts.

### `eagerEsModules`

- **Type:** `boolean`
- **Default:** `true`

Use ES module `import` statements instead of `require()` for artifact imports. When enabled, `graphql` tagged templates are replaced with an identifier and a corresponding `import` statement is added to the top of the file.

```js
relay({
  eagerEsModules: false,
})
```

### `outputFileExtension`

- **Type:** `'typescript' | 'javascript'`

Override the file extension used for artifact imports. Takes precedence over `language`.

### `artifactDirectory`

- **Type:** `string`

Custom artifact directory (relative to `rootDir`). When set, artifact imports resolve to this directory instead of being co-located with source files.

### `projects`

- **Type:** `ProjectConfig[]`

Multi-project configuration for monorepos with multiple Relay projects. Each project can specify its own `rootDir` and `artifactDirectory`.

```js
relay({
  projects: [
    { rootDir: './packages/app', artifactDirectory: './packages/app/__generated__' },
    { rootDir: './packages/admin', artifactDirectory: './packages/admin/__generated__' },
  ],
})
```
